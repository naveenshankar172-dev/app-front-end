# ManuFormat

Automated Document Formatting System  
Domain: IT Solutions

## Overview
ManuFormat is a web-based IT solution that automates academic document formatting.
It reduces manual effort and ensures consistent document structure.

## Features
- Upload raw document
- Automated formatting
- Clean and responsive UI
- Download formatted document

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Document Processing: python-docx

## System Workflow
User → Upload → Processing → Formatting → Download

## Future Enhancements
- AI/NLP-based section detection
- Multiple document templates
- PDF export support
